Telhas de atalho no Github Desktop:

Commit:

Ctrl + Enter: para commitar

Ctrl + P: para dar push no commit

Abrir o repositório:

Ctrl + Shift + A: para abrir o diretorio no code;

Ctrl + Shift + F: para abrir no explorer do windows;

Ctrl + Shift + G: para abrir no site Github.

Criar novo repositório:

Ctrl + N
